# 2026-04-17 项目进展记录

## 一、当前项目做到哪里了

截止今天，这个项目已经从“单片机本地功能开发”推进到了“软硬件最小闭环联调已打通”的阶段。

当前已经打通的链路是：

1. STM32 采集温湿度
2. OLED 本地显示温湿度与继电器状态
3. STM32 通过串口上报 telemetry
4. 电脑侧 Python 桥接脚本读取串口数据
5. 桥接脚本把数据转发到 FastAPI 后端
6. 后端生成控制命令
7. 桥接脚本把命令通过串口发回 STM32
8. STM32 执行命令并返回 ACK
9. 桥接脚本把 ACK 状态回写给后端

也就是说，**最小软硬件闭环已经跑通**。

### 3. 串口口线已经确认

通过 KE1 资料与实际联调，已经确认：

- `TO-MCU` 模式下，MCU 和 NB 模组通信走的是 `USART3 (PB10/PB11)`
- 板载 USB 到电脑侧可见的 MCU 串口实际走的是 `USART1 (PA9/PA10, 115200)`

所以当前 STM32 固件串口已经切换到 **`USART1`**。
